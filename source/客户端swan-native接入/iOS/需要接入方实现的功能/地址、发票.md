## 功能说明
地址、发票功能指 `swan.chooseAddress` 和 `swan.chooseInvoiceTitle` 能力以及和宿主配套的地址、发票管理中心，管理中心一般包含添加、删除、编辑等能力

使用示例：外卖小程序，填写配送地址时，可以调用 `swan.chooseAddress`能力直接选择宿主管理的地址，获取信息后直接填写相关信息

> 收货地址 BBAMNPShippingAddressProtocol

```
/*!
 *    @brief    请求信息
 */
@class BBAMNPRequestInfo;

/*!
 *    @brief    小程序对应的开发者ID信息
 */

/*!
 *    @brief    收货地址接口
```

* 1、添加收货地址

```
/*!
 *    @brief    获取用户添加收货地址的请求信息

 *                     l2 = 市id;
 *                     l3 = 区/县 id;
 *                     name = 收货人;
 *                     phone = 联系电话;
 *                     street = 街道地址 ;
 *                     zipcode = 邮编;
 *                   }
 *    @param     identify     小程序对应的开发者ID信息
 *    @param     requestInfoCallback 请求信息 回调
 */
+ (void)requestInfoForCreate:(NSDictionary *)postData
                     idntify:(BBAMNPAppIdentifyInfo *)identify
                    callBack:(void(^)(BBAMNPRequestInfo *info,NSError *error))requestInfoCallback;
```

> 示例 以添加收货地址为例

```

#import <Foundation/Foundation.h>
#import "BBAMNPShippingAddressProtocol.h"

NS_ASSUME_NONNULL_BEGIN

@interface BBAMNPShippingAddressImplement : NSObject <BBAMNPShippingAddressProtocol>

@end

NS_ASSUME_NONNULL_END


#import "BBAMNPShippingAddressImplement.h"
#import "BBAMNPRequestInfo.h"
#import "BBAMNPAppIdentifyInfo.h"

#define BBAMNPShippingAddressCallBackRequestInfo(url)\
BBAMNPRequestInfo *requestInfo = [BBAMNPRequestInfo reqestInfoWithUrl:url params:[[NSMutableDictionary alloc]initWithDictionary:postData]];\
if (requestInfoCallback) {\
requestInfoCallback(requestInfo,nil);\
}

// 用户添加收货地址
static NSString *SHOPPING_ADDRESS_CREATE_URL = @"ma/address/create";

@implementation BBAMNPShippingAddressImplement
+ (nonnull NSString *)requestURLForCreate {
    return [self requestURLWithPath:SHOPPING_ADDRESS_CREATE_URL];
}

/// 用户添加收货地址 address/create
+ (void)requestInfoForCreate:(NSDictionary *)postData
                     idntify:(BBAMNPAppIdentifyInfo *)identify
                    callBack:(void(^)(BBAMNPRequestInfo *info,NSError *error))requestInfoCallback{
    BBAMNPShippingAddressCallBackRequestInfo([self requestURLForCreate])
}

+ (NSString *)rootServerHost {
    return [BBAMNPPlatformService rootServerHost];///宿主的服务器地址
}

+ (NSString *)composeParameters {
    return [BBAMNPPlatformService composeParameters];///宿主服务器请求所需的公参
}

+ (NSString *)requestURLWithPath:(NSString *)path{
    
    NSString *urlString = [[NSString alloc] initWithFormat:@"%@/%@?%@", [self rootServerHost], path, [self composeParameters]];
    return urlString;
}
@end
```
