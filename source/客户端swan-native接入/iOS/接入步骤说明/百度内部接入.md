```

```
#### 1.3.1.3. 添加小程序初始化代码
* 在appDelegate中的`didFinishLaunchingWithOptions`方法中 添加小程序初始化代码。如下：

```
/// appID为宿主app在iTunes上的appID
 [SWANManager registerAppID:@"appID" launchOptions:launchOptions];
        
```
