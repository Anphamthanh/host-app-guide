#开发配置插件

配置插件以npm包形式提供,入口文件需返回一个object,实现以下方法:

    export {
        // 场景值列表,格式如下
        getSenceList: async () => [],
        // 根据swanNativeVersion,extensionjs来获取宿主app版本
        getHostAppVersion: extensionJsVersion => 'appVersion',
        // 提供当前native版本支持的extensionJs版本范围
        getExtensionJsRange: swanNativeVersion => {
            min: '',
            max: ''
        }
    }
